Project Title

Dashboard UI – coursefinder.ai

Project Description

This project is a responsive web-based dashboard interface built using HTML, CSS, Bootstrap 5, jQuery, and JavaScript.
It represents an admin/agent dashboard for an education consultancy platform (coursefinder.ai), showcasing applications, statistics, events, news, and quick navigation links.

The layout includes a collapsible sidebar, top navigation bar, interactive tables, carousels, and cards for displaying data in a clean and modern UI.
